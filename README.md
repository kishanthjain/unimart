# UniMart – Student Marketplace

UniMart is a student-to-student marketplace designed for college campuses. It allows students to buy, sell, and discover second-hand items such as books, electronics, cycles, hostel essentials, fashion products, and more.

## Features

* Buy and sell items within your college community
* Search products by keyword
* Filter products by category and college
* Wishlist support
* Cart management
* Orders page
* Seller dashboard
* Product detail page
* Responsive design for desktop and mobile
* Local storage based data management

## Categories

* Academic Books
* Electronics
* Hostel Essentials
* Cycles / Transport
* Fashion
* Fitness

## Installation

1. Clone the repository

```bash
git clone https://github.com/kishanthjain/unimart.git
```

2. Open the project folder

```bash
cd unimart
```

3. Open `index.html` in your browser.
